MT5 Connection Setup
Setup Instructions
  1. In MT5 go to Tools → Options → Expert Advisors
  2. Enable Allow WebRequest for listed URL and add: http://127.0.0.1:8080/api/mt5/trade
  3. Set MT5_BRIDGE_KEY in .env to match the EA's API_KEY input parameter
  4. Attach GalacticBridge.mq5 to any chart for each account
  5. Restart the FastAPI server: python main.py
